# 数字

## 函数

### parseInt()

```ts
function parseInt(value: string | number, radix: number): number;
```

转为数字类型

#### 参数

##### value

转换的值

`string` | `number`

##### radix

`number` = `10`

进制数，默认10进制

#### 返回

`number`

返回转换后的数字

#### Description

解决部分浏览器在转换 '08','09'等是0开头时被默认转8进制问题
