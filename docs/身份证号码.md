# 身份证号码

## 函数

### getInfoByIDCard()

```ts
function getInfoByIDCard(idCard: string): 
  | {
  province: string | undefined;
  birthday: string | undefined;
  age: number | undefined;
  sex: string | undefined;
}
  | undefined;
```

根据身份证号码获取信息

#### 参数

##### idCard

`string`

身份证号码，支持一代15位和二代18位

#### 返回

  \| \{
  `province`: `string` \| `undefined`;
  `birthday`: `string` \| `undefined`;
  `age`: `number` \| `undefined`;
  `sex`: `string` \| `undefined`;
\}
  \| `undefined`

返回身份证信息对象

#### Description

能获取到 籍贯，出生日期，年龄，性别
