# 分页

## 函数

### getLimit()

```ts
function getLimit(pagination: {
  page: number;
  pageSize: number;
  total: number;
  totalPage: number;
  pagerCount: number;
}): number[];
```

获得分页起始数

#### 参数

##### pagination

分页参数

###### page

`number`

###### pageSize

`number`

###### total

`number`

###### totalPage

`number`

###### pagerCount

`number`

#### 返回

`number`[]

返回起始数

### getNextPage()

```ts
function getNextPage(pagination: {
  page: number;
  pageSize: number;
  total: number;
  totalPage: number;
  pagerCount: number;
}, callback: (page: number) => void): Promise<number>;
```

获得下一页

#### 参数

##### pagination

分页参数

###### page

`number` = `1`

###### pageSize

`number` = `10`

###### total

`number` = `0`

###### totalPage

`number` = `0`

###### pagerCount

`number` = `7`

##### callback

(`page`: `number`) => `void`

回调函数

#### 返回

`Promise`\<`number`\>

返回下一页

#### Description

最后一页会根据totalPage参数判断，如果不传此参数，则会根据参数中pageSize和total重新计算总页数
