### toFixed()

```ts
function toFixed(
   num: string | number, 
   decimals: number, 
   mode: number): string;
```

强制保留小数位数

#### 参数

##### num

数字

`string` | `number`

##### decimals

`number` = `2`

保留小数的位数，默认2位

##### mode

`number` = `MATH.ROUND`

保留小数模式

#### 返回

`string`

返回保留后的数字字符串

#### Description

默认保留两位小数，解决原生的toFixed()会五舍六入的问题

***
