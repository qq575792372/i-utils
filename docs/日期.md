# 日期

## 接口

### DateOptions

日期配置类型

#### 参数

日期格式字符串，支持的占位符如下：
 | 占位符 | 说明 | 示例（2025-01-05 08:05:08 周一） |
 |--------|-----------------------|---------------------------------|
 | yyyy | 4 位年份 | 2025 |
 | yy | 2 位年份 | 25 |
 | MM | 2 位月份（补 0） | 01 |
 | M | 1 位月份（不补 0） | 1 |
 | dd | 2 位日期（补 0） | 05 |
 | d | 1 位日期（不补 0） | 5 |
 | HH | 24 小时制（2 位，补 0） | 08 |
 | H | 24 小时制（1 位，不补 0）| 8 |
 | hh | 12 小时制（2 位，补 0） | 08 |
 | h | 12 小时制（1 位，不补 0）| 8 |
 | mm | 2 位分钟（补 0） | 05 |
 | m | 1 位分钟（不补 0） | 5 |
 | ss | 2 位秒数（补 0） | 08 |
 | s | 1 位秒数（不补 0） | 8 |
 | SSS | 3 位毫秒（补 0） | 123 |
 | S | 1 位毫秒（不补 0） | 1 |
 | E | 周几（迷你名） | 一（zh） / Mon（en） |
 | EE | 周几（短名） | 周一（zh） / Mon（en） |
 | EEE | 周几（全名） | 星期一（zh） / Monday（en） |
 | Q | 季度（迷你名） | 1（zh） / 1（en） |
 | QQ | 季度（短名） | Q1（zh） / Q1（en） |
 | QQQ | 季度（全名） | 第一季度（zh） / Quarter 1（en）|
 | a | 上午 / 下午（小写） | am /pm |
 | A | 上午 / 下午（大写） | AM / PM |
 | aa | 上午 / 下午（中文） | 上午 / 下午 |
 | AA | 上午 / 下午（中文大写） | 上午 / 下午 |

#### 示例

```ts
yyyy-MM-dd HH:mm:ss → 2025-01-05 08:05:08
yyyy-M-d h:m:s → 2025-1-5 8:5:8
yyyy-MM-dd EE → 2025-01-05 周一
```

#### 参数

语言，可选值：zh（默认）/ en

#### 属性

##### format?

```ts
optional format: string;
```

##### lang?

```ts
optional lang: string;
```

### fromDateUTC()

```ts
function fromDateUTC(date: Date): Date;
```

UTC 日期对象转本地时区日期对象

#### 参数

##### date

`Date` = `...`

UTC 日期对象，默认当前 UTC 时间

#### 返回

`Date`

本地时区日期对象（时间戳 = UTC时间戳 + 时区偏移毫秒数）

#### Description

将 UTC 时间的 Date 对象转换为本地时间的 Date 对象（时间戳不变，仅调整时区偏移）

#### 示例

```ts
// UTC时间：2025-01-26 00:00:00
const utcDate = new Date('2025-01-26T00:00:00Z');
const localDate = fromDateUTC(utcDate);
console.log(toDateString(localDate)); // 2025-01-26 08:00:00（北京本地时间）
```

### toDateUTC()


#### 参数

##### date

`Date` = `...`

本地日期对象，默认当前本地时间

#### 返回

`Date`

UTC 日期对象（时间戳 = 本地时间戳 - 时区偏移毫秒数）

#### Description

将本地时间的 Date 对象转换为 UTC 时间的 Date 对象（时间戳不变，仅调整时区偏移）

#### 示例

```ts
// 北京本地时间：2025-01-26 08:00:00（东8区）
const localDate = new Date('2025-01-26 08:00:00');
const utcDate = toDateUTC(localDate);
console.log(utcDate.toISOString()); // 2025-01-26T00:00:00.000Z（UTC时间）
```
