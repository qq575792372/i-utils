# 函数

## 函数

### debounce()

```ts
function debounce<T>(
   fn: T, 
   delay: number, 
immediate: boolean): (...args: Parameters<T>) => void | ReturnType<T>;
```

防抖函数

#### 类型参数

##### T

`T` *extends* (...`args`: `any`[]) => `any`

#### 参数

##### fn

`T`

目标函数

##### delay

`number` = `1000`

延迟时间，单位毫秒，默认 1*1000 毫秒

##### immediate

`boolean` = `true`

是否立即执行，默认true

#### 返回

返回function()

```ts
(...args: Parameters<T>): void | ReturnType<T>;
```

##### 参数

###### args

...`Parameters`\<`T`\>

##### 返回

`void` \| `ReturnType`\<`T`\>

#### Description

事件执行后，在延迟时间内如果再次执行，会清空定时器重新延迟执行，举例：用户在输入框进行输入搜索，最终是会获取到最后一次输入，节约请求资源

***

### throttle()

```ts
function throttle<T>(fn: T, interval: number): (...args: Parameters<T>) => void | ReturnType<T>;
```

节流函数

#### 类型参数

##### T

`T` *extends* (...`args`: `any`[]) => `any`

#### 参数

##### fn

`T`

目标函数

##### interval

`number` = `1000`

时间间隔，单位毫秒，默认1000毫秒

#### 返回

返回function()

```ts
(...args: Parameters<T>): void | ReturnType<T>;
```

##### 参数

###### args

...`Parameters`\<`T`\>

##### 返回

`void` \| `ReturnType`\<`T`\>

#### Description

高频触发时，在指定时间间隔内只执行一次，举例：监听页面滚动，不会频繁触发，只会在固定时间内获取一次

***

### sleep()

```ts
function sleep(delay: number): Promise<unknown>;
```

睡眠延迟执行

#### 参数

##### delay

`number` = `1000`

延迟时间，单位毫秒，默认1000毫秒

#### 返回

`Promise`\<`unknown`\>

#### Description

需要配合 async/await 来达到延迟效果
