# 键盘

## 函数

### getKeyName()

```ts
function getKeyName(keycode: number): string;
```

根据keycode获得键名

#### 参数

##### keycode

`number`

键值

#### 返回

`string`

返回键名

***

### getKeyCode()

```ts
function getKeyCode(keyname: string): string | undefined;
```

根据keyname获得键值

#### 参数

##### keyname

`string`

#### 返回

`string` \| `undefined`

返回键值
