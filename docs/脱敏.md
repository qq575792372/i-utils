# 脱敏

## 函数

### formatStartOf()

> **formatStartOf**(`value`, `start`, `len`): `string`

指定位置的字符串转为星号

#### 参数

##### value

`string`

字符串参数

##### start

`number`

转换开始位置

##### len

`number`

显示星号的长度

#### 返回

`string`

返回转化后字符串
