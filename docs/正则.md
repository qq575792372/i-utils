# 正则

## 函数

### regexpTest()

```ts
function regexpTest(value: string, regex: RegExp): boolean;
```

正则校验的方法

#### 参数

##### value

`string`

校验的参数

##### regex

`RegExp`

使用的REGEXP中的正则

#### 返回

`boolean`

返回校验的结果

#### Description

类型为REGEXP对应的正则
