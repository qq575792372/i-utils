### zeroStart()

> **zeroStart**(`value`, `maxLength`): `string`

数字前补齐0达到指定位数

#### 参数

##### value

`string`

补零的数字

##### maxLength

`number` = `2`

补齐0后的最大长度，默认2位

#### 返回

`string`

返回补0后指定位数的字符串

#### Description

相当于原生的 padStart(2,'0')
