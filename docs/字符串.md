### inString()

```ts
function inString(value: string, str: string): boolean;
```

#### 参数

##### value

`string`

包含的元素

##### str

`string`

查找的字符串

#### 返回

`boolean`

返回true和false

### padZeroStart()

```ts
function padZeroStart(value: string | number, maxLength: number): string;
```

数字前补齐0达到指定位数

#### 参数

##### value

补零的数字

`string` | `number`

##### maxLength

`number` = `2`

补齐0后的最大长度，默认2位

#### 返回

`string`

返回补0后指定位数的字符串

#### Description

相当于原生的 padStart(2,'0')

***
