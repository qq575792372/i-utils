# 随机数

## 函数

### getRandom()

> **getRandom**(`min`, `max`): `number`

生成指定大小的随机整数

#### 参数

##### min

`number` = `0`

随机数的最小值，默认 0

##### max

`number` = `9`

随机数的最大值，默认 9

#### 返回

`number`

返回指定大小的随机整数

#### Description

n和m参数表示最小和最大范围值，默认0-9之间范围

***

### getRandomDigit()

> **getRandomDigit**(`len`): `number`

生成固定位数的随机整数

#### 参数

##### len

`number` = `1`

固定的位数

#### 返回

`number`

返回固定位数的随机数

#### Description

默认是1，代表生成0-9之间一位，如果是2，则生成10-99之间两位，以此类推
