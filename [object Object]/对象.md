### getValueByPath()

> **getValueByPath**(`target`, `path`): `any`

根据字符串属性路径获取目标对象的值

#### 参数

##### target

`Record`\<`any`, `any`\>

目标对象

##### path

`string` = `"data"`

字符串属性路径

#### 返回

`any`

返回目标对象

#### 示例

```ts
let res = {code:200, data:{rows:[], pages:{current:1,pageSize:20}}}
this._getValueByPath(res, 'data.pages.pageSize'); // 这里会输出20
```
